# Sikevux’s dotfiles

## Installation

```bash
CURDIR=$(pwd)
pusd ~
ln -s ${CURDIR}/.emacs.d/init.el ~/.emacs.d/init.el
ln -s ${CURDIR}/.zshrc ~/.zshrc
popd
```
